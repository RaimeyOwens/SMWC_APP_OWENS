# SMWC_APP_OWENS

Final project for Rize course web development. Creating a campus life app. A web application that addresses common pain points in college life through a unified interface. This project allows you to showcase the web development skills you've learned throughout this course while building something useful for the college community.



The SMWC Campus Life Web App is a student-focused website designed to help users explore life at Saint Mary-of-the-Woods College. It highlights campus locations, athletics, events, student organizations, and helpful resources to improve student engagement and community involvement.

This project was built as a multi-page front-end web application using HTML, CSS, Bootstrap, JavaScript, and a weather API integration.


## Pages Included

### Home (Index Page)
- Overview of campus life
- Key campus locations:
  - Le Fer Hall
  - Dining Hall
  - Knoerle Gym
- Contact form for student inquiries
- Helpful departments tab section

---

### Athletics Page
- Overview of SMWC athletics program
- Workout center (Club 64)
- Main gym information
- Student fan section
- Club sports (Esports, Cheer, Cycling, Dance, Powerlifting, Rowing)
- Equestrian program section
- Campus map and Google Maps embed

---

### Events Page
- Campus engagement information
- Student organizations
- Event participation opportunities
- Campus photo gallery
- Weather API

---

## Features Included In This App

### Weather API Integration
- Live weather data using OpenWeatherMap API
- Displays:
  - Temperature (°F)
  - Weather conditions
  - Weather icon
- Helps students plan daily campus activities

---

###  UI Features
- Bootstrap responsive design
- Carousel image sliders
- Collapsible information sections
- Navigation bar with dropdown menu
- Smooth scrolling effects
- Glyphicons for icons
- Grid-based layout system
